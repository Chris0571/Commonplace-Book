  
 docker ps -a                      (List container)

$ docker image                      (List images)

$ docker top                        (Running process)

$ docker push/pull {image}          (Scaricare e caricare immagini)

$ docker run -i -t ubuntu /bin/bash (Lanciare un'immagine nel terminale)

$ docker start/stop                 (Manage container)

$ docker rm {container id}          (Remove container)
