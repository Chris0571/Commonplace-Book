  
$ docker ps -a                      
					(List container)

$ docker image ls / images          
					(List images)

$ docker top {container id}         
					(Running process)

$ docker push/pull {image}          
					(Scaricare e caricare immagini)

				    	(Lanciare un'immagine nel terminale)

$ docker start/stop {container id}  
					(Manage container)

$ docker rm {container id}          
					(Remove container)
